## What is TRC - Sistema de Viajes?

TRC is a travel management system built for organizations that need to coordinate employee transportation between multiple branch offices. It provides a centralized platform where administrators can manage trips, employees, carriers, and branches — all in one place.

## Key Features

- **Role-based access control** — Two roles: Gerente de Tienda (admin) and Usuario, each with different access levels
- **Trip scheduling** — Create trips linking a collaborator, branch office, carrier, cost, and distance
- **Distance validation** — Trips enforce a maximum route of 150 km
- **Cost tracking** — Each trip records its cost; zero-cost trips are rejected
- **Reports** — View a full travel report joining users, branches, and carriers
- **Session management** — Secure session-based authentication via `express-session`
- **MySQL database** — Backed by MySQL with Sequelize ORM and transactional writes

## Tech Stack

| Layer | Technology |

| Sessions | express-session |

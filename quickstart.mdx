## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js** v18 or later
- **MySQL** 5.7+ or **MariaDB** 10.4+
- **Git**

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/molina071/TRC.git
    cd TRC
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    npm install
    ```

    This installs Express, Sequelize, EJS, bcrypt, express-session, and the MySQL2 driver.
  </Step>

  <Step title="Create and import the database">
    Open your MySQL client and create the database, then import the bundled SQL dump:

    ```bash
    mysql -u root -p -e "CREATE DATABASE sistema_viajes;"
    mysql -u root -p sistema_viajes < BD/sistema_viajes.sql
    ```

    <Note>The SQL file is located at `BD/sistema_viajes.sql` in the project root. It creates all tables and inserts seed data.</Note>
  </Step>

  <Step title="Configure the database connection">
    Open `config/Conexion.js` and update the credentials to match your MySQL setup:

    ```js
    const { Sequelize } = require('sequelize');
    const sequelize = new Sequelize('sistema_viajes', 'root', 'your_password', {
        host: 'localhost',
        dialect: 'mysql',
    });
    module.exports = sequelize;
    ```

    <Warning>The default configuration uses an empty password. Update the third argument if your MySQL root account has a password.</Warning>
  </Step>

  <Step title="Start the application">
    ```bash
    node app.js
    ```

    You should see:

    ```
    Server is running on port 3000
    ```
  </Step>

  <Step title="Log in">
    Open your browser and navigate to [http://localhost:3000](http://localhost:3000).

    Use one of the seed accounts to log in:

</Steps>
